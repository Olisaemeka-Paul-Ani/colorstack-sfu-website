# ColorStack SFU

Official website for the ColorStack SFU chapter — a community for
underrepresented students in computing at Simon Fraser University.

## Stack

- React + TypeScript
- Vite
- Framer Motion

## Running locally

```bash
npm install
npm run dev
```

Then open the local URL Vite prints in the terminal (defaults to
http://localhost:5173).
